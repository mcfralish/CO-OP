Section 3 – Description of technology used for your site

In this project I created a pretty standard Flask app. Flask is a python framwork that I am familiar with having created a few other apps with this software. I am hosting this site on heroku, as well as using their database service. I am using Flask SQLAlchemy to interpret SQL Queries via Python. I used jQuery and Bootstrap for formatting and effect.
