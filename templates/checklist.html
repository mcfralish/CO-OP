Feature Components

I used the database to store and retrieve User information as well as on the Cooperatives page to store Worker Cooperative info.

I used AJAX on my site in several way. Firstly, I used it on each page to load the header. This allowed me to load a different version depending on whether or not the user is authenticated without repeating the logic in each html document. Additionally, I used AJAX extensively on my Cooperatives page. I used it to to either load the form to create a new Cooperative if the user is authenticated. I also used AJAX to allow the user to filter their view of the Cooperatives by category by loading a different query.

My site features information about worker cooperatives and uses a consistnat theme and color pallete throughout.

My site uses flask as a framework, which is a bit more than a library. It also uses flask sql alchemy for db management. I also used some other Python utility libraries.

This site uses baseline javascript functions and the jQuery library to retrieve the header bar information.

This site uses a Member's Only area on the Cooperatives page where member's can add additional co-ops to the list of listed co-ops.
